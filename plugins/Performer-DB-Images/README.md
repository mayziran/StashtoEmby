# Performer DB Images

**版本：1.0.0**

在 Stash 演员详情页中浏览外部来源图片，并支持将选中的图片直接设置为演员主图。

## 功能

- 在演员详情页增加 `DB 图片` 按钮
- 按演员已有的 `stash_ids` 自动识别可用来源
- 按来源分别加载图片，互不影响
- 支持分页浏览和大图预览
- 支持直接将外部图片设置为演员主图
- 支持预览状态下的键盘操作

## 使用前提

- 当前演员需要至少绑定一个可用的外部 `stash_id`
- 对应的外部来源需要已在 Stash 中正确配置

## 安装

1. 将 `Performer-DB-Images` 文件夹放入 Stash 插件目录
2. 重启 Stash
3. 在 **设置 -> 插件** 中确认插件已加载

## 使用方法

1. 打开演员详情页
2. 点击 `DB 图片`
3. 选择要浏览的来源
4. 点击缩略图进入预览
5. 点击 `设为演员图片` 将当前图片设置为演员主图

## 界面说明

- 来源按钮：按来源分别加载图片
- `(…)`：该来源正在加载
- `(数字)`：该来源已加载完成，并显示图片数量
- `(!)`：该来源加载失败
- `每页`：设置当前来源每页显示数量，配置保存在浏览器本地

## 键盘操作

仅在预览层打开时生效：

- `Left`：上一张
- `Right`：下一张
- `Esc`：关闭预览

## 注意事项

- 插件不会自动创建外部 ID，只使用演员当前已有的 `stash_ids`
- 图片数量和可用性取决于外部来源实际返回结果
- 某个来源加载失败不会影响其他来源使用
- 当前不提供图片比例筛选和质量排序
