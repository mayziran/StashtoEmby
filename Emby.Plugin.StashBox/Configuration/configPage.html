StashBox

支持多个 Stash-Box 实例的外部 ID 插件。
包括：StashDB、ThePornDB、FansDB、JAVStash、PMV Stash 以及自定义实例

默认的 Stash-Box GraphQL 端点
例如：https://stashdb.org/graphql
已知的 Stash-Box 实例列表，每行一个
插件会自动识别这些端点并跳转到对应的网站
例如： 
https://stashdb.org/graphql
https://theporndb.net/graphql
https://fansdb.cc/graphql
https://javstash.org/graphql
https://pmvstash.org/graphql
